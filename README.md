## Object-Oriented Programming (OOP) Lab
This repository contains the implementation of core OOP concepts using Java. The primary goal of this lab is to demonstrate how to structure code into reusable, modular components that model real-world entities and behaviors.
## Core Concepts Covered
The implementation in OOPLab.java focuses on the four main pillars of Object-Oriented Programming:
* Encapsulation: Bundling data (attributes) and methods (functions) into a single unit (class) while restricting direct access to some components using access modifiers (private, public, protected).
* Inheritance: Creating a hierarchy where a "child" class acquires the properties and behaviors of a "parent" class, promoting code reusability.
* Polymorphism: Allowing objects to be treated as instances of their parent class, specifically through Method Overriding (runtime) and Method Overloading (compile-time).
* Abstraction: Hiding complex implementation details and showing only the necessary features of an object using abstract classes and interfaces.
## Technical Highlight
* Class Design: Defined custom classes to represent specific objects.
* Constructors: Used to initialize objects with specific states upon creation.
* Super & This Keywords: Proper utilization of keywords to navigate class hierarchies and reference current instances.
*Access Modifiers: Strategic use of private and public to ensure data security.
## Getting Started
### Prerequisites
* JDK (Java Development Kit) 8 or higher.
* A text editor (VS Code, Notepad++) or a full Java IDE.
  ### Compilation & Execution
   1. Clone the repository:
   ```Bash```
   git clone https://github.com/Haniatahir05/OOP-LAB.git
   ```Bash```
   2. Navigate to the project directory:
   ```Bash```
   cd OOP-LAB
   ```Bash```
   3. Compile the file:
   ```Bash```
   javac OOPLab.java
   ```Bash```
   4. Run the application:
   ```Bash```
   java OOPLab
   ```Bash```
